Starting with OCaml Batteries Included

Installation

Several methods exist for installing OCaml Batteries Included:

  • The GODI package manager is rather complicated to install for the first time but is then quite simple to use. It works under any version of Unix, including Linux, MacOS X, BSD, ...
  • Doing it by hand is quite easy to get started, but you probably want some real package management in the long run.
  • Batteries and OCaml can be installed with the ocamlbrew tool, this will work in fewer environments but should be easier than GODI in the environments it works in.
