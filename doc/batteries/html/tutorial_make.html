Installing OCaml Batteries (and OCaml) by hand

If you have access to a Unix like shell you can quickly install both Ocaml and Batteries using their own installation scripts. This is suffisient to get you started if you only want a taste of OCaml and/or Batteries, but will become uneasy as time goes by since installing new libraries will require step by step manual operations while upgrading anything will prove very painfull.

instructions given here are tailored for OCaml 3.12.0 and Batteries 2.0 but should work more or less with more recent versions.

Installing Ocaml

Let's say you want to install you system into $PREFIX: you@host:~$ PREFIX=/some/dir you@host:~$ BINDIR=$PREFIX/bin you@host:~$ LIBDIR=$PREFIX/lib you@host:~$ MANDIR=$PREFIX/man you@host:~$ SRCDIR=$PREFIX/src you@host:~$ DOCDIR=$PREFIX/share/docs you@host:~$ mkdir -p $BINDIR $LIBDIR $SRCDIR you@host:~$ _

Now download OCaml v3.12.0: you@host:~$ wget -P $TMPDIR 'http://caml.inria.fr/pub/distrib/ocaml-3.12/ocaml-3.12.1.tar.bz2' you@host:~$ tar jxf $TMPDIR/ocaml-3.12.1.tar.bz2 -C $SRCDIR you@host:~$ _

Now build ocaml: you@host:~$ cd $SRCDIR/ocaml you@host:ocaml$ ./configure --prefix $PREFIX you@host:ocaml$ make world.opt you@host:ocaml$ [sudo_if_required] make install you@host:ocaml$ _

Installing findlib/OUnit

Download and untar: you@host:~$ wget -P $TMPDIR http://download.camlcity.org/download/findlib-1.2.7.tar.gz you@host:~$ tar zxf $TMPDIR/findlib-1.2.7.tar.gz -C src you@host:~$ wget -P $TMPDIR http://forge.ocamlcore.org/frs/download.php/495/ounit-1.1.0.tar.gz you@host:~$ tar zxf $TMPDIR/ounit-1.1.0.tar.gz -C src you@host:~$ _

Build and install: you@host:~$ cd src/findlib-1.2.7 you@host:findlib-1.2.7$ ./configure -bindir $BINDIR -mandir $MANDIR -sitelib $LIBDIR/ocaml/site-lib you@host:findlib-1.2.7$ make all you@host:findlib-1.2.7$ make opt you@host:findlib-1.2.7$ [sudo?] make install you@host:findlib-1.2.7$ cd ../ounit-1.1.0 you@host:ounit-1.1.0$ ocaml setup.ml -configure you@host:ounit-1.1.0$ ocaml setup.ml -build you@host:ounit-1.1.0$ [sudo?] ocaml setup.ml -install you@host:ounit-1.1.0$ _

Installing Batteries

Proceed essentially the same for batteries: you@host:~$ wget -P $TMPDIR 'https://github.com/ocaml-batteries-team/batteries-included/tarball/v2beta1' you@host:~$ tar zxf $TMPDIR/ocaml-batteries-team-batteries-included-*.tar.gz -C $SRCDIR you@host:~$ _

Now build Batteries itself: you@host:~$ cd $SRCDIR/ocaml-batteries* you@host:ocaml-batteries$ export DOCROOT=$DOCDIR/batteries you@host:ocaml-batteries$ make all doc you@host:ocaml-batteries$ [sudo?] make install install-doc you@host:ocaml-batteries$ _

You now have enough installed to get a first taste of batteries. You should add $BINDIR to your $PATH.
